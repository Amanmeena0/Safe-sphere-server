## Overview

Safe-Sphere is an open-source community safety platform that integrates a Flask backend with a React frontend to deliver real-time crime analysis, emergency reports, and incident reporting.  
Designed for developers, it offers a scalable and secure architecture to build impactful safety applications.

### Why Safe-Sphere?
This project was developed to create robust safety solutions with:

- **API-Driven Data Management**: Well-defined endpoints for user authentication, incident reporting, and geo-data.
- **Geospatial Crime Mapping**: Interactive maps visualizing police stations and crime hotspots.
- **AI Chatbot Support**: Intelligent conversational agents for guidance and information dissemination.
- **Modular UI Components**: Customizable React components for dashboards, reporting stations, and alerts.
- **Seamless Frontend Styling**: Tailwind CSS, modern UI, and responsive design.
- **Secure Data Management**: Role-based access control and protected endpoints supporting community safety initiatives.
